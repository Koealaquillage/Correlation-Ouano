Hello ! Ici c'est un script et ses librairies d'utilisation. Celui-ci sert à analyser les données de Cristèle pour le lagon de Ouano. Il fait notamment des analyses de décalage temporel et de corrélation.

Le code est largement commenté, n'hésitez pas à le lire. Il utilise de façon intensive la librairie panda (qui n'est pas incluse ici).
